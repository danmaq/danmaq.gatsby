# Gatsby theme for danmaq.com

## Install

### Mac, Linux, or BSD

```SH
$ yarn install
```

### Windows

1. Run on Administrator.

```PowerShell
PS:Admin> yarn global add windows-build-tools
PS:Users> yarn install
```
